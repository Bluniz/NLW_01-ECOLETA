# 🔥Projeto - Como usar?
## 💻Backend
Para instalar a API,
 abra o terminal dentro da pasta backend/server e digite:
 
 ```sh
 #Intalar as dependências
 npm install
```
```sh
#Rodar as Migrates e Seeeds
npm run knex:migrate
npm run knext:seed
```
```sh
#Iniciar o servidor
yarn dev
```
Este projeto está configurado para rodar na porta 3333!
